Sequential Search

A simple way to search through and array.

Sample code for sequential sort:

public int sequentialSort(int [] array, int value) {
    for (int index=0; index<array.length; index++) { //loop through all elements to find the target value
        if (array[index] == value) { //if the current value being searched is equal to the target value
            return index; //return the index of the current value
        }
    }
    return -1; //if not found, return this
}

Sequential search is simple. Given a value to search, the sort simply traverses from left to right and checks every index to see if it matches the value being searched for. If an index does match, that index is returned, indicating that there is a value in that index. If the index doesn’t match, a value indicating that the value being searched for is not in the array is returned. Said value is usually -1, since it is impossible for an array to have an index of -1. The search is currently checking index 0. It finds that 6 is not equal to 3, so it moves on. 

The search is currently checking index 1. It finds that 3 is equal to 3, so it returns 1, the index where 3 is found. Note that there is another 3 at index 5, but that wasn’t returned because a value of 3 was already found in index 1.

Since the search can potentially span the entire array, the time complexity for this search is O(N). This means that the number of operations this process takes can go up to N, which is the size of the array.
