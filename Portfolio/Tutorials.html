Overview

In the ICS courses, we learned several important concepts. The most important of these concepts are explained in these tutorials. 

OOP is a style of programming that relies on the creation of multiple objects interacting with each other. This style can greatly improve the organization of your code, and can help simplify othewise complicated code.

Arrays are an essential part of programming in any language. They allow for the storing of multiple values, be it simple integers or complex classes, in a single variable. This is important for logic flow of the program.

Thought ArrayLists have a similar function to arrays, there are many crucial differences, the most important of which is this: ArrayLists don't have a set length, while arrays do!

Searching and sorting is another important aspect of programming. After all, it's how data is managed in programs. These tutorials cover 2 searching methods and 3 sorting methods.
